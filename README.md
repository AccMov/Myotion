# Myotion
#

> ## environment setup
> First of all, Clone this project to your disk
> 
> ### Windows
> 1. Install Anaconda from https://www.anaconda.com/download
> 2. open Anaconda power shell
> ```console
> cd %PATH_OF_THIS_PROJECT%
> conda env create --name accmov --file=accmov.yaml
> ```
>
> ## run
> open Anaconda power shell
> ```console
> cd %PATH_OF_THIS_PROJECT%
> conda activate accmov
> python main.py
> ```
